# Adjutorix Product Command Deck

The Product Command Deck converts Adjutorix's mounted authority
surfaces into one controlled product experience.

## Product behavior

All authority surfaces remain mounted. Their contracts, event
handlers, workspace bridges, verification logic, and runtime markers
remain intact.

The product shell suppresses uncontrolled fixed-overlay fanout and
allows exactly one authority surface to become visible at a time.

## Operator entry points

- Select **Power** from the primary workbench toolbar.
- Use **Command–Shift–P** to toggle the deck.
- Search all mounted authority powers.
- Filter by current chain, verification, archive, finality,
  certificate, publication, or authority.
- Select a surface to activate it.
- Press **Escape** to return to the main workbench.

## Product invariant

Hidden does not mean removed.

The command deck changes presentation authority only. It does not
delete, unmount, rewrite, or bypass any evidence, governance,
verification, finality, archive, or publication authority.
